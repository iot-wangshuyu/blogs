## 注意

该主题能渲染`首页(/)`，`文章页(/archives/${postUrl})`，`单个标签页(/tags/tagUrl)`，`自定义页面(/p/${postUrl})`。

## 使用方法

克隆或者[下载](https://github.com/ruibaby/vno-halo/releases)，将文件夹名改为`vno`，压缩为zip压缩包之后在后台的主题设置直接上传即可使用。
